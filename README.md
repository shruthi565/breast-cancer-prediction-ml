# ML Model Comparison Project

This project compares the performance of different machine learning models on the Breast Cancer dataset.

## Models Used
- Random Forest
- XGBoost
- LightGBM

## Dataset
Scikit-learn Breast Cancer dataset

## Steps
- Data loading
- Train-test split
- Model training
- Accuracy comparison

## Output
Comparison of model accuracies

## How to Run
```bash
pip install -r requirements.txt
python model_comparison.py
